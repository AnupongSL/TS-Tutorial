Any คืออะไร
- any คือ การนิยามตัวแปรที่มีชนิดข้อมูลแบบใดก็ได้ กล่าวคือ ตัวแปรใดๆที่ TypeScript ไม่สามารถบอกได้ว่าเป็นข้อมูลแบบใด
  TypeScript จะกำหนดข้อมูลนั้นให้เป็น any

ตัวอย่างการใช้ any 
- let myVar : any = "anupong"; //เป็นสตริง
  myVar = 10; // เป็น number
  myVar = true; // เป็น boolean

ประโยชน์ของ any 
- การกำหนด any นั้นจะสามารถทำได้เมื่อต้องการอยากให้ข้อมูลมีการเปลี่ยนแปลงค่าได้

ข้อควรระวัง
- ควรระบุชนิดข้อมูลให้เรียบร้อย ไม่ควรปล่อยให้มีชนิดข้อมูลเป็น any เพราะอาจทำให้เกิดข้อผิดพลาดได้
  เนื่องจาก any นั้นสามารถทำให้ค่าข้อมูลเปลี่ยนเนื่องจากไม่มีการตรวจสอบชนิดของข้อมูล
  ถ้าเลี่ยงไม่ได้ให้เปลี่ยนจาก any ไปใช้ unknown แทน

Unknown คืออะไร
- มีลักษณะคล้ายกับ any คือ การนิยามตัวแปรที่มีชนิดข้อมูลแบบใดก็ได้ แต่ก็จะมีการตรวจสอบชนิดข้อมูลก่อนเริ่มทำงาน

ตัวอย่างการใช้ Unknown
- let myVar: unknown = "anupong" //เป็นสตริง
  if(typeof myVar === "number"){
    // กระบวนการทำงาน
  }

สรุป Any & Unknown 
- ตัวแปรของเราสามารถเปลี่ยนชนิดข้อมูลได้โดย
- any ไม่มีการตรวจสอบชนิดข้อมูล
- unknown มีการตรวจสอบชนิดข้อมูล
