tsconfig.json คืออะไร
- คือไฟล์สำหรับตั้งค่าการทำงานของ TypeScript Compiler ประกอบด้วยค่าพื้นฐานที่ควรทราบดังนี้
- 1. target
    "traget":"es2016" เป็นการกำหนดให้คอมไพล์ TypeScript ออกมาเป็น JavaScript es2016
- 2. rootDir
    "rootDir:"./" ตำแหน่งโฟลเดอร์ที่เก็บกลุ่มไฟล์ TypeScript(.ts)
- 3. outDir
    "outDir":"./dist/" ตำแหน่งโฟลเดอร์ที่เก็บกลุ่มไฟล์ JavaScript(.js) ที่คอมไพล์เรียบร้อยแล้ว เช่น โฟลเดอร์ที่ชื่อว่า dist เป็นต้น
- 4. include
    "include":["src"] กำหนดขอบเขตของไฟล์ที่จะให้คอทไพล์ เช่น ไฟล์ที่ถูกจัดเก็บในโฟลเดอร์ src เป็นต้น
